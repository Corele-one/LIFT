# LIFT 电梯控制系统

这是一个基于FPGA的简单两层电梯控制系统，使用Verilog HDL实现。系统模拟了一个两层电梯的基本功能，包括楼层请求、电梯运行状态显示和蜂鸣器提示。

## 功能特性

- **两层电梯模拟**：支持1楼和2楼之间的运行。
- **键盘输入**：通过4x4矩阵键盘输入楼层请求。
- **状态显示**：使用七段数码管显示当前楼层和运行状态（上升、下降、空闲、关机）。
- **LED指示**：16个LED灯指示楼层请求状态。
- **蜂鸣器提示**：电梯到达楼层时发出蜂鸣声。
- **开关控制**：开关控制电梯的开关机。

## 文件结构

- `elevator_top.v`：电梯控制系统的顶层模块，实现状态机和整体逻辑。
- `keypad.v`：矩阵键盘控制器模块，处理键盘扫描和按键编码。
- `dynamic_led6.v`：动态七段数码管显示模块，驱动六个数码管显示楼层和状态。
- `uut.xdc`：Xilinx FPGA的引脚约束文件，定义IO端口的电气标准和引脚分配。

## 系统要求

- **FPGA开发板**：支持Xilinx FPGA（如Artix-7系列），具有足够的IO引脚。
- **开发工具**：Xilinx Vivado Design Suite（推荐版本2018.1或更高）。
- **时钟频率**：默认50MHz，可通过参数调整。

## 构建和部署

1. **创建Vivado项目**：
   - 打开Vivado，创建新项目。
   - 选择合适的FPGA器件（根据您的开发板）。

2. **添加源文件**：
   - 将`elevator_top.v`、`keypad.v`、`dynamic_led6.v`添加为设计源文件。
   - 将`uut.xdc`添加为约束文件。

3. **综合和实现**：
   - 运行综合（Synthesis）。
   - 运行实现（Implementation）。
   - 生成比特流文件（Generate Bitstream）。

4. **下载到FPGA**：
   - 使用Vivado的硬件管理器连接到FPGA开发板。
   - 编程FPGA器件。

## 使用方法

1. **上电**：系统上电后处于关机状态。
2. **开机**：按下开关sw0开机，显示器显示"ON"。
3. **楼层请求**：
   - 按键0：1楼上升请求
   - 按键3：1楼内部请求
   - 按键12：2楼下降请求
   - 按键15：2楼内部请求
4. **观察状态**：
   - 数码管显示当前楼层（1或2）和状态（U上升、D下降、-空闲）。
   - LED灯指示未处理的请求。
5. **关机**：按下开关sw11关机，显示器显示"OFF"。

## 参数配置

- `CLK_FREQ_HZ`：系统时钟频率，默认50MHz。
- `SCAN_HZ`：键盘扫描频率，默认1000Hz。
- `DEBOUNCE_FRAMES`：键盘去抖帧数，默认5。
