# 模块化电梯控制系统 (Modular Elevator Control System)

本项目是一个基于 FPGA 的模块化电梯控制系统实现，使用 Verilog 语言编写。相比于之前的单模块版本，本项目将核心逻辑、外设控制和显示驱动进行了拆分，提高了代码的可读性和可维护性。但是还没有上板过，仅用作实验报告的编写。

## 项目结构

项目采用自顶向下的设计方法，主要模块如下：

- **`elevator_modular_top.v`**: 顶层模块，负责所有子模块的实例化和信号连接。
- **`elevator_controller.v`**: 核心控制逻辑模块，实现电梯的状态机（FSM），处理楼层请求、运行方向和状态转换。
- **`clock_divider.v`**: 分频器模块，产生系统所需的 0.1s 定时脉冲。
- **`keypad.v` (matrix_keypad_controller)**: 4x4 矩阵键盘控制模块，包含扫描逻辑和消抖处理。
- **`elevator_motion_view.v`**: 显示预处理模块，将电梯状态转换为 7 段数码管和 LED 动画所需的显示数据。
- **`dynamic_led6.v`**: 6 位数码管动态扫描驱动模块。
- **`buzzer_controller.v`**: 蜂鸣器控制模块，负责电梯到达时的提示音逻辑。

## 功能特性

1.  **多层控制逻辑**: 目前支持 2 层楼的电梯逻辑模拟。
2.  **电源控制**: 通过 `sw0` 开关控制系统启动（POWER ON）和关闭（OFF）。
3.  **强制复位**: 通过 `sw11` 开关可将电梯强制复位至 1 楼。
4.  **矩阵键盘输入**:
    - 支持内部选层按键和外部召唤按键。
    - 具备硬件消抖功能，确保按键触发准确。
5.  **可视化显示**:
    - **数码管**: 实时显示当前楼层、运行状态（上行 `A` / 下行 `d` / 停止 `0`）以及运行倒计时。
    - **LED 阵列**: 模拟电梯运行的上下行流水灯动画。
6.  **声音提示**: 到达指定楼层时，蜂鸣器发出间歇性提示音。

## 状态机说明 (FSM)

电梯核心逻辑包含以下状态：
- `ST_OFF`: 系统关闭。
- `ST_POWER_ON`: 系统上电自检/初始化。
- `ST_IDLE_1F`: 在 1 楼待机。
- `ST_IDLE_2F`: 在 2 楼待机。
- `ST_UP`: 向上运行。
- `ST_DOWN`: 向下运行。

## 管脚分配简表 (Xilinx Vivado)

| 信号名称 | FPGA 管脚 | 说明 |
| :--- | :--- | :--- |
| `clk` | D4 | 50MHz 时钟输入 |
| `rst_n` | T8 | 异步复位 (低电平有效) |
| `sw0` | T9 | 系统电源开关 |
| `sw11` | F3 | 强制复位至 1 楼 |
| `col[3:0]` | T10, R11, T12, R12 | 矩阵键盘列输入 |
| `row[3:0]` | R10, P10, M6, K3 | 矩阵键盘行输出 |
| `seg[7:0]` | (见 XDC) | 数码管段码 |
| `dig[5:0]` | (见 XDC) | 数码管位选 |
| `led[15:0]` | (见 XDC) | 状态与动画 LED |
| `beep` | L2 | 蜂鸣器输出 |

## 如何运行

1.  将 `多模块` 文件夹下的所有 `.v` 文件添加至 Vivado 或 Quartus 工程。
2.  导入 `uut.xdc` 约束文件。
3.  编译、综合并下载至开发板。
4.  拨动 `sw0` 开启电梯，使用键盘进行选层。
